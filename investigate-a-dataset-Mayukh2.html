Introduction¶

In order to investigate how genre impacts a movie, I have downloaded the TMDb movie data cleaned by Kaggle. Each row in this dataset has information regarding a single movie. Primary fields related to our analysis in this row are:

  • Title
  • Genres
  • Release Date
  • Budget
  • Revenue

In this report, I explore the following:

  • How has the popularity of genres changed over the years. This kind of a reflection on the chaning moods of the movie viewing public

  • Since this dataset has the adjusted budget and revenue details, I want to explore which genre of movies has the highest/lowest share in the top 100 grossing movies of all time

# A lot of columns are pipe delimited values, lets create a function to split the data to get the values as a list
# Input to the function is a dataframe column and the split character
def parse_pipedata(c, s):
    split_list = c.str.split(s)
    ret_list = list()
    for row_index, row in enumerate(split_list):
        for col_index, item in enumerate(row):
            ret_list.append(item)

    #Remove duplicates
    ret_list = list(set(ret_list))
    ret_list.sort()
    
    #Return the list
    return ret_list

Find the Genre which is trending highest and lowest in terms of popularity over the years¶

We want to create a dataframe which has the release year and the genre of movie and the average popularity of all movies belonging to that genre for that year. In order to compute a trend, this needs to be a rolling average for 5 years¶

# For each genre 
for g in movie_genres:
    #Filter all records for that Genre of Movie
    g_df = df[df['genres'].str.contains(g)==True]
    
    #Group By Year and find the Average Popularity of the Genre by Year
    mean_pop_df = g_df.groupby(['release_year'], as_index=False)['popularity'].mean().rename(columns={'popularity':g+'_Avg'})
    
    #Calculate the rolling average popularity of the Genre by Year
    mean_pop_df[g+'_rolling_avg'] = mean_pop_df[g+'_Avg'].rolling(5).mean()
    
    #Drop the column which is just the Average Popularity
    mean_pop_df = mean_pop_df.drop(g+'_Avg', 1)
    
    #Merge columns for each iteration to the dataframe created for Genres
    df_genre_popularity = pd.merge(df_genre_popularity, mean_pop_df, on='release_year', how='left')

#Sort and replace NaN with 0's
df_genre_popularity.sort_values('release_year', inplace=True)

# For the ones that didnt have any movies in that year, this will turn up with NaN, so lets replace that by 0
df_genre_popularity.fillna(0, inplace=True)

Conclusions¶

The Adventure Genre is right now the trending the highest in terms of popularity. Westerns seem to have really fallen out of favor.

Even from the top 100 grossing movies of all time, its evident that the maximum number of them are of genre Adventure.

So Pro Tip for a new film-maker, make it an Adenture movie.
